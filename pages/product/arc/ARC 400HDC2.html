Product Overview

Virgo Plus has specially configured this ARC welding machine to achieve excellent welding results using advanced IGBT module and fast recovery module with high-frequency inverter technology. The ARC 400HDC2 ensures large output power with stable and reliable performance for demanding industrial applications.

This machine features ARC/LIFT TIG function, VRD function control, current & voltage digital display, and a remote control box for adjusting welding current at your fingertips. The unique ARC LENGTH control function helps the welder perform precision pipe welding even at curve points.

Key Advantages

IGBT Module Technology

Adopting IGBT module and fast recovery module with high-frequency inverter technology for large output power, stable and reliable performance.

Remote Control

Remote Control Box allows adjustment of welding current at your fingertips for greater efficiency and convenience on-site.

Wide Electrode Compatibility

OCV of 85 supports E6010, E7010 and E8010 electrodes, unlocking wider electrode use — ideal for all-position pipe welding.

ARC LENGTH Control

Unique ARC LENGTH control function helps the welder perform pipe welding at curve points with precision and ease.
