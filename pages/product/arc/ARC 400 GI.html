Product Overview

Virgo Plus has specially configured this ARC welding machine to achieve excellent welding results using advanced IGBT module inverter technology. The ARC 400 G is a professional-grade 3-phase industrial welder featuring a specially designed Capacitor PB that enables operation on diesel generators, making it ideal for remote and field applications.

This machine adopts the latest inverter technique with pulse width technology, ensuring high working efficiency while maintaining a compact and lightweight design. The full wave with full control Bridge delivers pure DC wave output, and the separate current and volt meter provides precise real-time monitoring of welding parameters.

Key Advantages

Diesel Generator Compatible

Specially designed Capacitor PB allows the machine to operate reliably on diesel generators for remote and field welding applications.

Separate Meters

Dedicated current and volt meters for precise, real-time monitoring of welding parameters during operation.

Remote Control Option

Standard provision for remote option with switch provides convenient control from a distance during welding operations.

VRD Safety Feature

VRD Provision reduces voltage during non-welding periods, significantly enhancing operator safety against electric shock.
