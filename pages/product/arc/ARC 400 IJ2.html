Product Overview

Virgo Plus has specially configured this ARC welding machine to achieve excellent welding results using advanced IGBT module inverter technology. The ARC 400 IJ2 is a professional-grade 3-phase inverter welder with full wave full control bridge for pure DC wave output, delivering exceptional welding performance.

This machine features adjustable ARC force for improved arc stability, separate current and volt meter for precise monitoring, and heavy duty lug type output terminals. With a wide current adjustment range of 40–400A, VRD provision for safety during non-welding, and standard provision for Remote option with Switch, the ARC 400 IJ2 is built for demanding industrial applications.
